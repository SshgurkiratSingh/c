### Headings
To create a heading, you use the hash symbol (`#`) before your heading text. The number of `#` symbols used will determine the level of the heading.

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

### Emphasis
You can make text italic or bold.

```markdown
*This text will be italic*
_This will also be italic_

**This text will be bold**
__This will also be bold__

*You **can** combine them*
```

### Lists
Markdown allows you to create ordered and unordered lists.

```markdown
Unordered list:
- Item 1
- Item 2
  - Item 2a
  - Item 2b

Ordered list:
1. Item 1
2. Item 2
3. Item 3
```

### Links and Images
To add links and images, you use a specific syntax.

```markdown
[This is a link](http://example.com)

![Alt text for an image](http://example.com/image.jpg)
```

### Code Blocks
For inline code, use single backticks (`), and for code blocks, use triple backticks (```). Optionally, you can specify the programming language immediately after the opening backticks for syntax highlighting.

```markdown
Inline `code` has `back-ticks around` it.

```python

    return "Hello, Markdown"

```

### Blockquotes
Blockquotes are created using the greater than sign (`>`).

```markdown
> This is a blockquote.
```

### Horizontal Rules
To create a horizontal rule, use three or more asterisks (`***`), dashes (`---`), or underscores (`___`).

```markdown
---
```
